# MeshBridge NoteBoard Frontend

React + Vite 前端應用

## 安裝依賴

```bash
npm install
```

## 開發模式

```bash
npm run dev
```

## 建置生產版本

```bash
npm run build
```

建置後的檔案會輸出到：
- `templates/app_noteboard/index.html` - HTML 模板
- `static/app_noteboard/` - JavaScript 和 CSS 資源

## 目錄結構

```
frontend_app_noteboard/
├── src/
│   ├── App.jsx          # 主要應用組件
│   ├── main.jsx         # 應用入口
│   └── index.css        # 全局樣式
├── index.html           # HTML 模板
├── vite.config.js       # Vite 配置
└── package.json         # 依賴管理
```
